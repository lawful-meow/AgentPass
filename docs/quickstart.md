# AgentPass Quickstart

Choose one of the quick initialization methods below:

1. **Via curl:** `curl -fsSL https://raw.githubusercontent.com/lawful-meow/AgentPass/main/setup_agentpass.py | python3 - --inject`
2. **Via uv:** `uv run https://raw.githubusercontent.com/lawful-meow/AgentPass/main/setup_agentpass.py --inject`

Then use `/checkpoint` to save state and `/resume latest` to load.
